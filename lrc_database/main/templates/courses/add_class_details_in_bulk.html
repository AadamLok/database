Add Class Details in Bulk

Write class data into the text area below. For each class you wish to add, add a single line with the following values in order and separated by commas.

  • full course ID, like "1"
  • class day, like "Monday"
  • class time, like "14:00"
  • class location, like "ILC S311"
  • class duration, like "01:15"
so the full line would be something like 1,Monday,14:00,ILC S311,01:15

You can export large numbers of lines like this easily from programs like Excel by looking for an "export to CSV" (comma-separated values) or similar option.
