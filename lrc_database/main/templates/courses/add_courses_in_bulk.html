Add Courses in Bulk

Write course data into the text area below. For each course you wish to add, add a single line with the following values in order and separated by commas.

  • department short name, like "COMPSCI"
  • course number, like "198C"
  • course name, like "Programming in C"
so the full line would be something like COMPSCI,198C,Programming in C

You can export large numbers of lines like this easily from programs like Excel by looking for an "export to CSV" (comma-separated values) or similar option.
