Create users in bulk

Write user data into the text area below. For each user you wish to add, add a single line with the following values in order and separated by commas.

  • an email address, like "jshmoe@umass.edu"
  • a first name, like "joe"
  • a last name, like "shmoe"
so the full line would be something like jshmoe@umass.edu,joe,shmoe

Note: Capitalizations doesn't matter, and all the users that would be created during this process will be automatically added to Staff group.

You can export large numbers of lines like this easily from programs like Excel by looking for an "export to CSV" (comma-separated values) or similar option.
